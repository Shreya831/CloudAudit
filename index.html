Create the CloudLens IAM Role

  1. In the AWS Console, go to IAM → Roles → Create role
  2. Select Another AWS account, enter Account ID: 730335564775
  3. Check Require external ID and paste your User ID as the external ID
  4. Attach the ReadOnlyAccess policy and Billing (read-only)
  5. Name the role CloudLensReadOnlyRole and create it
